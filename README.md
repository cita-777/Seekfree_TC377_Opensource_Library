### 工程结构说明

- 后缀开头`文件/文件夹`均为配置文件
- `code`为用户代码文件夹
- `Debug`为ADS编译产生文件
- `libraries`存放`逐飞科技/SeekFree`的开源库
- `user`存放**所有核心的main入口**及**总配置头文件**
